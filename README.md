# Fashion Styling App

A Java-based smart wardrobe management system that helps users organize clothing items and generate outfit recommendations through an interactive graphical interface.

## Features

- Manage clothing items by category, season, and color
- Outfit recommendation logic based on compatibility rules
- Dynamic filtering using Java Collections and lambda expressions
- Persistent data storage using file input/output
- Interactive GUI built with JavaFX

## System Architecture

The application follows an object-oriented design:

- Interface: "Wearable"
- Abstract class: "Clothing"
- Concrete classes: "Top", "Pants", "Shoes", "Bag", "Accessory"
- Management module: "Wardrobe"
- Recommendation module: "StyleRecommender"
- File handling module: "FileHandler"
- GUI module: "FashionStylingGUI"

## Technologies

- Java
- JavaFX
- Object-Oriented Programming
- Generics and Collections
- Lambda Expressions 
- File I/O 

## How to Run

1. Compile all Java files
2. Run "Main.java" or launch "FashionStylingGUI.java"
3. Manage wardrobe items and generate outfit suggestions
